# 新增功能实现说明

## 📋 概述
本次更新为漫画导入和管理功能添加了以下增强：

---

## 🎯 功能1：导入相关提升

### 1.1 拖放导入区域
**位置**：主窗口中心上方  
**功能**：用户可以直接将漫画文件夹拖入应用进行导入

**实现细节**：
- 新增 `DragDropArea` 类（main_window.py）
- 支持识别文件夹拖放事件
- 自动弹出导入对话框

**使用方式**：
```
1. 在拖放区域拖入漫画文件夹
2. 输入作者和标签（可选）
3. 点击确定完成导入
```

### 1.2 导入对话框增强
**功能**：
- 自动识别多集漫画中每一集的集数
- 在表格中显示各集信息
- 允许用户修改集数

**界面变化**：
- 新增"各集详细信息"表格
- 显示列：[文件夹名 | 自动识别集数 | 修改集数]
- 用户可直接编辑修改集数的值

**验证**：
- 检查集数是否重复
- 防止相同集数的导入冲突

### 1.3 集名识别与自定义
**识别规则**（见 `_EPISODE_PATTERN`）：
```
支持格式：
- 第5话、第5集、第5卷
- EP5、E5、e5
- [5]、(5)
- 001、0001 等纯数字
```

**自定义方式**：
- 在导入对话框的表格中直接修改集数
- 支持任意1-999之间的集数

### 1.4 导入错误日志
**功能**：导入失败时自动生成错误日志

**日志内容**：
```
漫画导入错误日志
============================================================

集数: 2
漫画ID: 1
错误信息: 该漫画集数已存在，请修改集数或先删除旧数据
------------------------------------------------------------
```

**生成位置**：
- 文件名格式：`error_[时间戳].log`
- 存储位置：导入源目录（拖放）或数据目录

**显示时机**：
- 导入完成后若有失败项
- 显示错误计数和日志路径
- 用户可在文本编辑器中查看详细信息

---

## 🎨 功能2：管理相关增强

### 2.1 集数管理标签页
**位置**：漫画属性对话框 → "集数管理"标签页

**功能**：
- 显示所有集数列表
  - 集数号（不可编辑）
  - 图片数量
  - 修改后的集数（SpinBox）
- 支持批量修改集数顺序
- 验证集数不重复

**使用方式**：
```
1. 右键漫画 → 编辑属性
2. 点击"集数管理"标签
3. 在表格中修改集数
4. 点击"确定"保存
```

**重排原理**：
- 使用临时编号避免唯一性约束冲突
- 先映射到负数，再映射到最终值
- 原子性事务保证数据一致性

### 2.2 封面管理标签页
**位置**：漫画属性对话框 → "封面管理"标签页

**功能**：
- 显示当前封面预览（140×190像素）
- 选择新封面图片
- 清空现有封面

**支持格式**：
- PNG、JPG、JPEG、BMP、GIF

**保存策略**：
- 自动复制选中图片到数据目录
- 生成时间戳防止覆盖
- 删除旧封面文件

**使用方式**：
```
1. 右键漫画 → 编辑属性
2. 点击"封面管理"标签
3. 点击"选择新封面"或"清空封面"
4. 点击"确定"保存
```

### 2.3 基本属性标签页（改进）
**保持原有功能**：
- 修改漫画名称
- 修改作者
- 修改标签
- 修改所属分组

**新增联动**：
- 集数重排后持久化
- 封面更新后立即生效

## 🐛 已知限制

1. 集数重排时所有新集数不能重复
2. 每次导入时重排需要手动操作
3. 拖放一次只能导入一个文件夹
4. 错误日志不支持按标准库logging保存

## 📝 使用示例

### 示例1：快速导入
```
1. 拖入 /path/to/漫画名
2. 输入作者名和标签
3. 点击确定 → 完成
```

### 示例2：修改集数
```
1. 导入后发现集3集4顺序错了
2. 右键 → 编辑属性 → 集数管理
3. 修改集3→4，集4→3
4. 点击确定 → 完成
```

### 示例3：处理导入错误
```
1. 导入时显示"跳过/失败 2 集"
2. 查看弹出的消息和错误日志
3. 修正错误原因后重新导入
```
