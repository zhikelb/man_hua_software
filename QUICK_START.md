**脚本位置**: `installer.nsi`

### 编译步骤
```powershell

#    下载: https://nsis.sourceforge.io/Download

# 2. 编译安装程序
"C:\Program Files (x86)\NSIS\makensis.exe" installer.nsi

# 3. 输出文件
#    生成: dist/漫画阅读器_installer.exe
```
