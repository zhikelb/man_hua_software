## 🛠️ 技术实现

### 代码结构
```
app/
├── main.py              # 应用入口和业务协调
├── config.py            # 配置管理
├── database.py          # 数据库模型和 ORM
├── services/
│   ├── import_service.py   # 导入业务逻辑
│   ├── library_service.py  # 书库业务逻辑
│   └── reader_service.py   # 阅读器业务逻辑
├── ui/
│   ├── main_window.py      # 主窗口
│   ├── import_dialog.py    # 导入对话框
│   └── reader_window.py    # 阅读器窗口
└── utils/
    ├── cover_generator.py  # 动态封面生成
    └── image_files.py      # 文件处理工具
```

## 🔄 后续维护指南

### 更新应用版本
```powershell
# 1. 修改源代码
# 2. 重新编译
venv\Scripts\pyinstaller build_exe.spec --distpath dist --workpath build --clean

# 3. 更新便携版本
Copy-Item -Path dist\漫画阅读器.exe -Destination dist_portable\ -Force
```

### 备份用户数据
```powershell
# 定期备份 data 文件夹
Copy-Item -Path dist_portable\data -Destination backup_$(Get-Date -Format yyyyMMdd) -Recurse
```
